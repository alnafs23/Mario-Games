PROJECT TITLE:
Mario Game in Greenfoot

PURPOSE OF PROJECT:
Membuat permainan sederhana bergaya Mario menggunakan Greenfoot.
Pemain mengendalikan karakter Mario untuk melompat, bergerak,
menghindari musuh, dan mengumpulkan poin.

VERSION or DATE:
Versi 1.0 - April 2026

HOW TO START THIS PROJECT:
1. Buka Greenfoot.
2. Import folder project Mario Game.
3. Klik tombol "Run" untuk menjalankan permainan.

USER INSTRUCTIONS:
- Gunakan tombol panah untuk bergerak ke kiri/kanan.
- Tekan spasi untuk melompat.
- Kumpulkan koin untuk menambah skor.
- Hindari musuh agar tidak kehilangan nyawa.
- Permainan berakhir jika nyawa habis.
